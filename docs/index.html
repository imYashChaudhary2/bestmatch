How it works
1
User types a device name
Type anything — "iPhone 16 Pro Max" or "Xiaomi Pad 5". Autocomplete shows 8 matching suggestions from the local DB.
2
Strict local DB lookup
Every word in the query must match the key exactly. Version tokens (m1, m2, 5, 6, pro, ultra) are never skipped — prevents wrong device matches.
3
AI fallback for unknown devices
If not in the local DB, Anthropic API returns structured JSON specs for any device ever made — unlimited coverage.
4
Specs render + image loads
Spec rows animate in immediately. Device image loads asynchronously from GSMArena CDN — 108 hardcoded verified URLs.
